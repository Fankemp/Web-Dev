# Онлайн-магазин (Лабораторная работа №4)

Учебный проект на Angular, имитирующий витрину товаров Kaspi.kz.

## Как запустить проект локально:

1. **Склонируйте репозиторий:**
   `git clone [ссылка_на_твой_репозиторий]`

2. **Перейдите в папку проекта:**
   `cd [название_папки]`

3. **Установите все зависимости:**
   `npm install`

4. **Запустите сервер:**
   `ng serve`

5. **Откройте браузер:**
   Перейдите по адресу `http://localhost:4200/`
