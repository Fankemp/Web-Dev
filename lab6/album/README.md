Учебный проект на Angular, имитирующий витрину товаров Kaspi.kz.

## Как запустить проект локально:

1. **Склонируйте репозиторий:**
   `git clone https://github.com/Fankemp/Web-Dev`

2. **Перейдите в папку проекта:**
   `cd C:\Users\s_musaev\Web-Dev\lab6\album`

3. **Установите все зависимости:**
   `npm install`

4. **Запустите сервер:**
   `ng serve`

5. **Откройте браузер:**
   Перейдите по адресу `http://localhost:4200/`
